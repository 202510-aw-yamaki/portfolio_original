【対象ファイル】
yh_portfolio/profile.html

【ページの目的】
- ドーナツ型の6セクションを入口に、プロフィール情報を段階表示する。
- 1画面内で「選択 -> 詳細確認 -> 閉じる」を完結させる。

【HTML構造】
- `.stage` 配下は `header / .main / footer.bottom-bar` の3層。
- `header`:
  - `.brand`（ロゴ + テキスト）
  - `nav`（4ページリンク、現在ページは `aria-current="page"`）
- `.main`:
  - `.donut-area`
    - `#instruction`（初期ガイド）
    - `svg.donut`（JSでセグメント動的生成）
  - `#dim.main-background-dim`（背景ディム）
  - `#panel.content-panel`（詳細パネル: `#panelTitle / #panelList / #panelNote / #closeBtn`）
  - `.center-placeholder`（レイアウト補助）
- `footer.bottom-bar`:
  - `.bottom-buttons` 内に4リンク。
  - 現在ページは `.is-current` + `tabindex="-1"`。

【JS仕様（inline script）】
- データ:
  - `sections` 配列（6項目）。各項目は `key / title / bullets / note?` を持つ。
  - `colors` と `activeColor` でセグメント色管理。
- SVG生成:
  - `polarToCartesian` / `describeArc` / `buildRingPath` でリングパス算出。
  - `createDonut()` が `path` とラベル `text` を各セクション分生成。
  - 各 `path` は `click` と `keydown(Enter/Space)` で `openPanel()` 実行。
- パネル表示:
  - `openPanel(section, pathEl)` でタイトル・箇条書き・注記を差し替え。
  - 選択セグメントのみ `activeColor`、それ以外は元色へ戻す。
  - `#instruction` を縮退、`#panel` を `.active`、`#dim` を `.visible`。
- パネル閉じる:
  - `#closeBtn` クリック / `#dim` クリック / `Escape` で `closePanel()`。
  - 配色・表示状態を初期へ復帰。
- 可変サイズ制御:
  - `fitDonutToMain()` で `.donut-area` 実寸に合わせ `--donut-size` を更新。
  - 中央ガイド円は `--instruction-size` と文字サイズ変数を連動調整。
  - テキスト溢れ時は段階的に縮小（最大20回ループでガード）。
  - `ResizeObserver` + `resize/orientationchange` で再計算。

【レスポンシブ/見た目】
- 共通配色変数は `css/common.css` を利用。
- ページ固有レイアウトは `css/profile.css`:
  - mobile: 全幅ステージ、中央ドーナツ主体。
  - tablet/pc: パディング・パネルサイズ・凡例位置を拡張。

【アクセシビリティ】
- セグメントは `role="button"` + `tabindex="0"`。
- パネルは `aria-live="polite"` と `aria-hidden` 切り替え。
- reduced motion は `common.css` の全体設定で短縮。
